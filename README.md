# Split21React

Проект ещё пишется. Это простенькая игра в 21-очко. Проект специально делается на react с минимальным количеством библиотек. Это игра с искуственным интелектом. Запросы отправляются на собвственный сервер, который проксирует запросы к groq api. Вся логика находится в react для улучшения навыков в его использовании. 

[Сервер](https://github.com/Kront2000/split21nodeServer) 

## Стек

- react
- pinia
- vite
- react-router
